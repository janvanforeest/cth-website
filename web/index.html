mijn git commands

git innit crëert een lege repository

git status geeft je een lijst met files die nog niet worden getracked en die werken nog niet

git add zorgt dat de gecrëerde files worden ge add zodat ze later officieel worden gemaakt met een commit

git branch crëert een andere stam om veranderingen in te plaatsen apart van de main branch

git remote kan helpen connecties te maken met andere repositories

git push stuurt de veranderingen die je gemaakt heb op je git worden gezien op je github

git clone maakt een copy van je repository

git pull zorgt dat als op github iemand anders een aanpassing heeft gemaakt en is dat niet op jouw computer geregristreerd dan kan je die info pullen naar je git zodat je upto date bent

git log geeft een lijst van alle commits die je net hebt gemaakt en of ze ook logisch zouden werken

git config zort ervoor dat er nieuwe configuraties worden gemaakt op dingen zoals text
